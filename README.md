# Ray-Tracer

C++ Whitted stye ray tracer

## Shading Model
* https://en.wikipedia.org/wiki/Phong_reflection_model

## Optimizations
* Multi-threading
* Kd-tree
* Adaptive/Jittered Anti-aliasing

## Fun Stuff
* SIRD images (red/cyan 3d images)

## Usage
* (mkdir build && cd build && cmake .. -DCMAKE_BUILD_TYPE=RELEASE && make -j8)
* run ./trace.sh --help
